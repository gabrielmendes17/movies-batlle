## Development

#### Pre-requisites:
* IntelliJ or vscode Lombok plugin
* Java 17
* Maven

#### Technologies:
* [Java](https://www.java.com/pt-BR/)
* [Maven](https://maven.apache.org/)
* [Spring](https://spring.io/)

#### Run application
Execute integration tests
```bash
$ make start
```

#### Integration Tests
Execute integration tests
```bash
$ make integration-tests
```
